# Инструкция по установке Link Manager Display плагина

## Шаг 1: Загрузите плагин на сайт

### Вариант A: Через FTP/cPanel
1. Создайте папку `/wp-content/plugins/link-manager-display/`
2. Загрузите файл `link-manager-display.php` в эту папку

### Вариант B: Через WordPress админку
1. Заархивируйте файл `link-manager-display.php` в ZIP архив
2. В WordPress админке: Плагины → Добавить новый → Загрузить плагин
3. Выберите ZIP файл и установите

## Шаг 2: Активируйте плагин
1. Плагины → Установленные
2. Найдите "Link Manager Display" 
3. Нажмите "Активировать"

## Шаг 3: Настройте виджет
1. Внешний вид → Виджеты
2. Найдите виджет "Link Manager Sidebar" (НЕ "Link Manager Links"!)
3. Перетащите в нужную область (Sidebar)
4. Откройте настройки виджета

## Шаг 4: Заполните настройки

### Для автоматического получения ссылок через API:
- **Title**: Links (или любое название)
- **API URL**: http://YOUR_SERVER:3000/api
  - Если backend на том же сервере: http://localhost:3000/api
  - Если на другом сервере: укажите правильный адрес
- **Site ID**: 3 (для theladyjersey.com)
- **API Key**: uviqmo0iylno1m77f9e6q
- **Manual Mode**: НЕ ставить галочку

### Для ручного режима (без API):
- Поставьте галочку "Manual Mode"
- Перейдите в меню "Link Manager" в админке
- Вручную добавьте ссылки

## Шаг 5: Сохраните и проверьте
1. Нажмите "Сохранить" в настройках виджета
2. Обновите главную страницу сайта
3. Ссылки должны появиться в sidebar

## Проблемы и решения

### Не появляются ссылки?
1. Проверьте, что backend запущен (http://YOUR_SERVER:3000/api/health)
2. Проверьте Site ID (должен быть 3 для theladyjersey.com)
3. Проверьте, что в базе есть размещенные ссылки для этого сайта

### Ошибка соединения с API?
1. Проверьте правильность API URL
2. Убедитесь, что сервер доступен с WordPress сайта
3. Проверьте файрволл/порты

### Нужно удалить старый плагин?
Да, удалите или деактивируйте старые версии:
- link-manager-receiver.php
- link-manager-receiver-enhanced.php
- Любые другие Link Manager плагины
