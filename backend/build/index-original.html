📥 WordPress Plugin Required

Version 2.2.0 • Updated: September 1, 2025

To connect your WordPress site, you need to install our Link Manager Widget plugin:

  1. Download the plugin: link-manager-widget.zip (v2.2.2)
  2. Install it on your WordPress site (Plugins → Add New → Upload Plugin)
  3. Activate the plugin
  4. Go to Settings → Link Manager Widget
  5. Copy the API key from there and use it below
API Endpoint: https://shark-app-9kv6u.ondigitalocean.app/api
